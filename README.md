### 数据库实验
#### 实验一～三 较为简单，实验四为综合实践（图书管理系统）
#### 实验任务一与实验任务二均在 Windows10 达梦7下完成，对应的 LabXCode.sql 中包含了建表、插入数据以及实验的所有代码。
#### 实验任务三与实验任务四均在 Arch Linux 系统下使用Qt和MySQL开发完成，环境配置和版本号如下。

#### 操作系统：Linux 4.10.13-1-ARCH
#### Qt Creator(Qt 的 IDE)：Qt Creator 4.3.0
#### Qt：Qt 5.8.0
#### GCC：GCC 6.3.1 20170306, 64 bit
#### MySQL：MySQL 14.14(运行在Ubuntu Server 14.04上)

#### 说明：由于实验任务三与四均在 Linux 下开发，因此若要运行需在 Windows 使用 Qt Creator 打开项目并重新编译
